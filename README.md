

# 🌍 Carbon Footprint Platform

A modern web application that helps users estimate and understand their environmental impact through carbon footprint calculations. The platform provides interactive insights and practical recommendations to encourage sustainable lifestyle choices.

## 🚀 Live Demo

🔗 https://carbon-footprint-platform-v2.vercel.app/

## 📖 About The Project

Carbon Footprint Platform is designed to raise awareness about climate change by helping users calculate their carbon emissions based on everyday activities. The application analyzes user inputs and provides a clear estimate of their environmental impact along with suggestions for reducing emissions.

The goal of this project is to combine technology and sustainability to promote informed and eco-friendly decision-making. Carbon footprint calculators are widely used to estimate the environmental impact of activities and lifestyles. :contentReference[oaicite:0]{index=0}

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/suraj-5021/carbon-footprint-platform-v2.git
```

### Install Dependencies

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
